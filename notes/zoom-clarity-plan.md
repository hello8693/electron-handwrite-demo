# 放大后模糊改进计划（方案 4 + 5 组合）

目标：放大时静态层保持清晰，同时不显著牺牲性能。

## 方案组合
1. **几何缓存驱动的高质量重烘焙（方案 4）**
   - 使用已有 stroke 点序列/宽度（几何缓存）重烘焙可见 tiles。
   - 通过动态 DPR 提升瓦片分辨率，不改变世界坐标对齐。

2. **抗锯齿策略（方案 5）**
   - 2D Canvas 在缩放时启用 `imageSmoothingEnabled`。
   - 对高缩放使用更高 tile DPR，减少放大模糊。

## 实施步骤
- [ ] S1：在渲染阶段启用 2D 图像平滑
- [ ] S2：基于 `viewport.scale` 动态调整 tile DPR（含阈值/上限）
- [ ] S3：在缩放/停止缩放时触发可见 tiles 的重烘焙
- [ ] S4：验证 tile 对齐与清晰度
